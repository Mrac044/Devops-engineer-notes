##  **Синтаксис правил**

```
# Игнорировать конкретный файл
config.local.yaml

# Игнорировать все файлы с расширением
*.log
*.tmp

# Игнорировать папку (включая все внутри)
node_modules/
.env/

# Исключение из правил игнорирования
!important.log

# Игнорировать во всех директориях
**/temp/

# Комментарии
# Это комментарий
```

### **Проверить, что игнорируется**

```bash

# Проверить игнорируется ли конкретный файл
git check-ignore -v путь/к/файлу

# Показать все игнорируемые файлы в статусе
git status --ignored
```
