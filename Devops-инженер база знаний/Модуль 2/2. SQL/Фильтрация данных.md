**`WHERE`** - оператор в языке SQL для отображения записей, соответствующих указанному условию

## Как работать с условиями выборки

Чтобы выбрать строки, которые нам нужны используют оператор `WHERE`. В запросе SELECT его нужно добавить и после него задать нужное нам условие.
#SQL-команда :

```sql
SELECT * FROM users WHERE id > 60;
```

[View on DB Fiddle](https://www.db-fiddle.com/f/7SmX3YRNShMaX9bK3etYcD/0)

В условиях мы можем использовать и другие операторы сравнения:

- `=` — равно
- `!=` — не равно
- `>` — больше
- `<` — меньше
- `>=` — больше либо равно, не меньше
- `<=` — меньше либо равно, не больше

#SQL-команда Текст в SQL должен записываться в одинарных кавычках. Правильный запрос будет выглядеть так:

```sql
SELECT * FROM users WHERE first_name = 'Johan';
```

[View on DB Fiddle](https://www.db-fiddle.com/f/7SmX3YRNShMaX9bK3etYcD/4)

#SQL-команда Следующий запрос выберет все курсы онлайн-школы, созданные начиная с 15 июня 2022 года:

```sql
SELECT name, created_at FROM courses WHERE created_at > '2022-06-14';
```

[View on DB Fiddle](https://www.db-fiddle.com/f/7SmX3YRNShMaX9bK3etYcD/5)

При использовании `WHERE` нужно помнить про тип данных столбца, по которому задается условие. Это важно, потому что для разных типов данных устанавливаются разные правила оформления: например, строки и даты должны быть заключены в одинарные кавычки.
