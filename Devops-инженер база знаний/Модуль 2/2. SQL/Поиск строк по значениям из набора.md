В некоторых ситуациях нам нужно найти не диапазон строк, а строки, в которых поле соответствует одному значению из набора.

## Как искать поля, соответствующие значению из набора

нужно найти пользователей с идентификаторами 1, 2 или 5. Если понадобится десяток совпадений, #SQL-команда можно использовать оператор `IN`. После него в скобках через запятую нужно указать значения, которым должно быть равно поле в запросе:

```sql
SELECT
    id,
    first_name,
    last_name
FROM users
WHERE id IN (1, 2, 5);
```

#SQL-команда Если нужно исключить определенные записи, достаточно добавить `NOT` перед `IN`:

```sql
SELECT id, first_name, last_name
FROM users
WHERE id NOT IN (1, 2, 5);
```

#SQL-команда Оператор `IN` можно использовать и с другими типами данных — например, со строками:

```sql
SELECT id, first_name, last_name
FROM users
WHERE first_name IN ('Lionel', 'Lucienne', 'Jennyfer');
```
