#SQL-команда Чтобы не задумываться о регистре букв в шаблоне можно воспользоваться оператором `ILIKE`, который ищет строки без учета регистра

```sql
SELECT id, first_name, last_name, email, birthday FROM users WHERE last_name ILIKE 'sch%t';
```
