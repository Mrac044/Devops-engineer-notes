#Реальная_работа На практике чаще встречаются динамические маршруты.

## Динамические маршруты

Чтобы разобраться, как выглядят динамические маршруты, проанализируем адреса некоторых курсов на Хекслете:

- [https://ru.hexlet.io/courses/php-introduction-to-oop](https://ru.hexlet.io/courses/php-introduction-to-oop)
- [https://ru.hexlet.io/courses/php-object-oriented-design](https://ru.hexlet.io/courses/php-object-oriented-design)
- [https://ru.hexlet.io/courses/js-react](https://ru.hexlet.io/courses/js-react)

В этих адресах прослеживается определенная структура: _/courses/<имя курса>_.
В примерах выше мы столкнулись с **динамическими маршрутами**. У таких маршрутов внутри изменяемые части, но обработчик у маршрута только один.

Например, все указанные выше адреса курсов соответствуют одному маршруту, который можно записать так: `/courses/{id}`. Где секция `{id}` означает, что на это место подставляется конкретный **идентификатор курса** — уникальная запись, которая отличает одну сущность от другой.

Сам способ записи зависит от конкретного фреймворка. В Flask для этого используются угловые скобки. Это создает ощущение, что используется интерполяция:

```python
@app.route("/courses/<id>")
def courses_show(id):
    return f"Course id: {id}"
```

```sh
curl localhost:8000/courses/132

Course id: 132
```

==Интерполяция в Python — это способ вставлять значения переменных внутрь строк. Например, с помощью f-строк можно легко вставлять переменные: `f"Значение: {переменная}"`==

## Плейсхолдеры

Любая изменяемая часть маршрута называется **плейсхолдером** — заполнитель. 

В маршруте выше только один плейсхолдер — `id`.

Для удобства пользователей в адресах стараются использовать не числовые идентификаторы, а текстовые названия. Например, вместо _/courses/332_ показывают _/courses/php-mvc_. Эту часть адреса называют [**Slug**](https://en.wikipedia.org/wiki/Clean_URL#Slug).

Flask позволяет в маршруте указать тип данных, которым может быть плейсхолдер как <тип данных>:

- `string`(по умолчанию) принимает любой текст без слешей
- `int`принимает только положительные целые числа
- `float`принимает положительные числа
- `path`подобен string, но также можно указывать слеш
 - `uuid`принимает UUID-строки

Количество плейсхолдеров в маршруте может быть больше одного. Обычно такие маршруты используются для вложенных ресурсов:

Количество плейсхолдеров в маршруте может быть больше одного. Обычно такие маршруты используются для вложенных ресурсов:

```python
@app.route("/courses/<int:course_id>/lessons/<int:lesson_id>")
def lessons_show(course_id, lesson_id):
    return f"Course id: {course_id}, Lesson id: {lesson_id}"
```

```sh
curl localhost:5000/courses/12/lessons/42

Course id: 12, Lesson id: 42
```
